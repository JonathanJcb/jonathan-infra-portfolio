\# Infrastructure Hybride Sécurisée



\## Contexte

Déploiement d’un SI hybride pour une PME multi-sites intégrant on-premise et cloud Azure.



\## Objectifs

\- Centraliser l’authentification

\- Sécuriser les accès

\- Automatiser les déploiements

\- Superviser la disponibilité

\- Mettre en place un PRA



\## Stack

Windows Server, Linux, VMware/Proxmox, Azure, Terraform, Ansible, Zabbix
